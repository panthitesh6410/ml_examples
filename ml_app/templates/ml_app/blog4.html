REINFORCEMENT MACHINE LEARNING

Definition of Reinforcement Machine-Learning


"Reinforcement Learning is a machine learning technique in which, there is a software agent taht makes observations and takes actions within an enviornment and in return it recieves rewards."

"It's objective is to learn to act in a way that will maximize its expected long-term rewards.
You can think of positive reward as a pleasure, and negative reward as pain.
The agent acts in the enviornment and learns by trail and error to maximize its pleasure and minimize it's pain."

Reinforcement Learning technique is widely used in game-play automation, simulation control, self-driving cars and many more areas.

EXAMPLES -

  • Agent can be the program controlling a walking robot.
    In this case the enviorment is the real world, the agent observes the enviornment through a set of sensors such as cameras and touch sensors, and its actions consists of sending signals to activate motors.
    It may be programmed to get positive rewards whenever it approaches target destination, and a negitive reward whenever it wastes time, or goes in the wrong dorection or falls down.
  • The agent can be the program playing board game, such as game of GO.
  • The agent can be the program controlling a video game like pacman.
    In this case, enviornment is a simulation, the actions are joystick positions, the observations are screenshots and rewards are game points.
