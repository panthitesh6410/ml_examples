SUPERVISED MACHINE LEARNING

Definition of Supervised Machine-Learning


In Supervised Machine Learning technique, we train the Machine Learning model using data which is well-labeled i.e. some data which is already tagged with correct output.
In other words, in supervised learning , the training data we feed to the algorithm includes the desired solutions called labels.
It can be said that, model is learning which takes place in the presence of a supervisor or teacher.

OR

A supervised machine learning model learns from labelled data(i.e. data which is already given and have correct output), helps to predict outcomes for unforeseen data.
It allows to produce output from previous experience.

Types of Supervised Machine Learning Techniques

There are 2 supervised machine learning strategies -

  1. Regression
  2. Classification

REGRESSION -

Regression technique predicts a single output value learning from training data. This is technique to predict continuous value.Regression works on comtinuous data( continuous data means integers, floating point values ).

Example -
  • To predict house proce based on parameters like locality, size of area, rooms etc.
  • Weather Forecasting
  • Predicting profit/loss based on expeditures
  • Predicting salary of an employee based on it's experience, age, qualification, etc.
    • and many more...

Classification -

Classification means to group or classify output into a particular class out of different possible classes. Classification works on discrete data.

Example -
  • To classify whether the cancer is begnin or malignant
  • To Predict whether the mail is spam or ham
  • To predict the species of a given flower
  • Predict whether a person is suffering from diabetes or not
    • and many more...
