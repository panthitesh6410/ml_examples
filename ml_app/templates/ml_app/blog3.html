UN-SUPERVISED MACHINE LEARNING

Definition of Un-supervised Machine-Learning


Un-supervised learning is a machine learning technique, where you do not need to supervise the model. Instead, you need to allow the model to work on it's own to discover information.
It mainly deals with unlablled data (i.e. data where final output is not present).

Un-supervised learning algorithms allows you to perform more complex preocessing tasks compared to supervised learning technique.
Although, un-supervised learning can be more unpredictable compared to other methods.

Un-supervised Machine Learning technique finds all kind of unknown patterns in data.

Types of Un-supervised Machine Learning Techniques

There are 3 Un-supervised machine learning strategies -

  1. Clustering
  2. Recommendation Systems
  3. Dimensionality Reduction

Clustering

Clustering mainly deals with finding a pattern or structure in a collection of unrecognized data.
Clustering technique process the data and find natural clusters or groups if they exists in data.

Examples -

  • Identifying buying patterns of customers in e-commerce
  • Recommending new products to customers in e-commerce
  • Credit card fraud detection
  • Identifying clusters of customers (like loyal, old, etc)
    • and many more...

Recommendation Systems -

A Recommender system refers to a system that is capable of predicting the future preference of a set of items for a user, and recommecnd the top items.

Examples -
  • Filpkart product recommendation system
  • Amazon product recommendation system
  • Netflix movie recommendation system
    • and many more...

Dimensionality Rediction -

Dimensionality Reduction refers to techniques that reduce the number of input variables in a dataset.
It helps in data compression, and hence storage space.
It reduces computation time.
It also helps to remove redundant features and data vizualization.
